Eruvin 2a עירובין ב׳ א
None מתני' מבוי שהוא גבוה למעלה מעשרים אמה ימעט ר' יהודה אומר None אינו צריך None והרחב מעשר אמות ימעט None ואם יש לו צורת הפתח אע"פ שהוא רחב מעשר אמות אין צריך למעט: None גמ' תנן התם None סוכה שהיא גבוהה למעלה מעשרים אמה פסולה ורבי יהודה מכשיר None מאי שנא גבי סוכה דתני פסולה וגבי מבוי תני תקנתא None סוכה דאורייתא תני פסולה מבוי דרבנן תני תקנתא None ואיבעית אימא None דאורייתא נמי תני תקנתא אלא None סוכה דנפישין מיליה פסיק ותני פסולה מבוי דלא נפישי מיליה תני תקנתא None אמר רב יהודה אמר רב None חכמים לא למדוה אלא מפתחו של היכל ורבי יהודה לא למדה אלא מפתחו של אולם None דתנן None פתחו של היכל גבהו עשרים אמה ורחבו עשר אמות ושל אולם גבהו ארבעים אמה ורחבו עשרים אמות None ושניהן מקרא אחד דרשו None (ויקרא ג, ב) ושחטו פתח אהל מועד None דרבנן סברי None קדושת היכל לחוד וקדושת אולם לחוד None וכי כתיב None פתח אהל מועד אהיכל כתיב None ורבי יהודה סבר None היכל ואולם קדושה אחת היא וכי כתיב None פתח אהל מועד אתרוייהו הוא דכתיב None ואיבעית אימא None לר' יהודה נמי קדושת אולם לחוד וקדושת היכל לחוד None והכא היינו טעמא דרבי יהודה דכתיב None אל פתח אולם הבית None ורבנן אי הוה כתב None אל פתח אולם כדקאמרת השתא דכתיב אל פתח אולם הבית הבית הפתוח לאולם None והא כי כתיב האי במשכן כתיב None אשכחן משכן דאיקרי מקדש ומקדש דאיקרי משכן None דאי לא תימא הכי הא דאמר רב יהודה אמר שמואל None שלמים ששחטן קודם פתיחת דלתות ההיכל פסולין שנאמר None (ויקרא ג, ב) ושחטו פתח אהל מועד בזמן שפתוחין ולא בזמן שהן נעולים None והא כי כתיב ההיא במשכן כתיב None אלא None אשכחן מקדש דאיקרי משכן ומשכן דאיקרי מקדש None בשלמא מקדש דאיקרי משכן דכתיב None (ויקרא כו, ד) ונתתי (את) משכני בתוככם None אלא משכן דאיקרי מקדש מנלן None אילימא מדכתיב None (במדבר י, כא) (ונשאו) הקהתים נושאי המקדש והקימו את המשכן עד בואם

Writing Helpful Reviews

For primary texts and historical translations: Indicate what outside source (book, web page) was used to compare the text, or make clear that your review is only looking at the text present.

I compared this text to X printed edition and found no problems.
I checked this text against the edition on X website, all looks good.
I read this text carefully and don't see anything wrong.

For original translations: Review the text carefully next to the original, and comment on its accuracy and style according to our translation guidelines.

This translation seems accurate to me.
The language could be improved, but the translation seems right.
This translation misunderstands the term X.
How to Score your Review:
Compared closesly to a source, confident there are no problems.
Compared to a source, didn't notice any problems.
Looks problem free, but could be reviewed more closely.
Found some issues that need to be addressed.
Found serious problems that need to be addressed.
